# Rydz App - Claude Code Instructions

## Project
Rydz is a ride-sharing mobile web app with rider, driver, and admin interfaces.

## Tech Stack
- HTML/CSS/JavaScript (vanilla)
- Static web app (no build step)

## File Structure
- `index.html` - Landing page
- `rider.html` - Rider interface
- `driver.html` - Driver interface
- `admin.html` - Admin dashboard
- `css/` - Stylesheets
- `js/` - JavaScript files
